- had/hadn't + 动词的过去分词
- 用法：当谈到过去，然后想说明比这个过去更早发生的事情时，用**过去完成时**。

<br><br/>

1) S9E15 The One With The Mugging

- 原句（过去完成时）：Ross: “…they took my backpack which had all the original artwork I had done…”  ￼
- 大意：他们抢走了我的背包，里面有我自己画的漫画原稿。
- 语境：Ross 讲自己小时候被抢的创伤经历，越说越悲壮，大家配合吐槽。 ￼
- 语法点：这里有两层“过去更早”：
	•	“they took my backpack”（一般过去）是叙述主事件；
	•	背包里“原稿已经在那儿/而且原稿早就画好了” ⇒ had / had done 用来把“更早发生/已完成”的信息压到背景里。

<br><br/>

2) S7E16 The One With The Truth About London

- 原句（过去完成时，被动）：Rachel: “…she had been had.”  ￼
- 大意：她被耍了/中了圈套。
- 语境：Ross 追 Ben 的路上摔下楼梯，Rachel 发现那是个假人（恶作剧），瞬间意识到“被整了”。 ￼
- 语法点：
	•	had been + 过去分词 是过去完成时被动。
	•	“had been had” 属于口语/俚语式表达（had = “骗/耍”），结构上仍是“在那个过去时刻之前，她已经处于‘被耍’的结果状态”。

<br><br/>

3) “My mom had just killed herself…”

句子：My mom had just killed herself…
关键在 had just：说明“更早那件事发生”离参照点非常近。

时间轴：
```
更早(P_before)            参照点(P_ref)                         NOW
妈妈刚去世(刚刚)    →    我14岁来到纽约/那阵子处境很惨    → 现在我讲给你听
 had just killed herself
```
你该感受到的变化：
“just”让镜头变成“贴脸回放”：不是遥远的更早过去，而是“刚刚发生”。

<br><br/>

- 一眼就能套用的“时间变化口诀”

- 当你想用过去完成时，脑子里先问一句：

- 我这句话里有没有两个过去？
	•	过去参照点（故事主镜头）：did  
	•	更早发生/已完成的背景：had done  
	•	更早且持续到参照点：had been doing
